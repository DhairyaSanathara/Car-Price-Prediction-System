# 🚗 Car Price Prediction System

## Overview

This project is my first End-to-End Machine Learning Deployment Project.

The application predicts the estimated resale value of a used car based on:

- Company
- Car Model
- Manufacturing Year
- Kilometers Driven
- Fuel Type

The trained Machine Learning model is deployed using Flask and can be accessed through a simple web interface.

## Machine Learning Workflow

1. Data Collection
